# Credit Scoring Model using Random Forest

## 📌 Project Overview

The Credit Scoring Model is a Machine Learning project developed to predict whether a customer is likely to have good or bad credit based on historical financial data.

The model uses the Random Forest Classification algorithm to analyze customer information and classify creditworthiness. This project demonstrates the complete machine learning workflow, including data preprocessing, model training, evaluation, and visualization.

---

## ✨ Features

- Load and analyze German Credit Dataset
- Check dataset information and missing values
- Split dataset into training and testing sets
- Train a Random Forest Classifier
- Predict customer creditworthiness
- Evaluate model performance
- Display Classification Report
- Display Confusion Matrix
- Calculate ROC-AUC Score
- Visualize ROC Curve
- Visualize Top 10 Important Features
